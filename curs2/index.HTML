"a" este un tag care defineste un hyperlink.

Cel mai important atribut al lui este "href" care indica o destinatie pt link.

Fara href "a" este doar un inlocuitor pentru un hyperlink.

Aici avem un exemplu de "a" cu "href" care va redirectioneaza la o pagina notorie pentru mine:

Gentoo AMD64 Handbook
